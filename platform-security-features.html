State-of-the-art security architecture

Zoomforth is built on a robust enterprise security architecture that provides you with the control you need to secure your users, content, and digital experiences.

Secure cloud hosting

We offer a world-class secure cloud hosting service from AWS (US) so your data is extremely safe.

Our servers sit within our own virtual private cloud (VPC) with exceptional cloud server security and network access control lists (ACLs) that prevent unauthorized requests getting to our internal network.

Prefer to self-host? No problem. Simply create your microsite and then export the site code to host on your own domain.

Wide range of access controls

Ensure your content reaches only those you intend, with a range of site security access controls.

Secure your microsites with a password, email authentication, multi-factor authentication or the security gold standard: Single Sign On.

User roles & permissions

Choose from a range of user roles and permission profiles to ensure your users only have access to the parts of the platform you want them to.
